## Run Project 
### Create a Virtual Environment
   ```bash
      # On Windows
      python -m venv env

      # On Linux/Mac
      python3 -m venv env
   ```
### Activate Virtual Environment
   ``` bash
      # On Windows
      .\env\Scripts\activate

      # On Linux/Mac
      source env/bin/activate
   ```
### Install Requirements
   ```bash

      pip install -r requirements.txt

### Run 
   ```bash
      python3 manage.py runserver
   ```
